# CodeGenerator
## 简介
基于 Freemarker &amp; Velocity 模板引擎的代码生成器。目前支持MySQL，Orancle、SQLServer。支持拓展。

## 使用
Java Application方式运行即可。
因为UI使用三方库，需要额外配置。需要在classpath中加入**beautyeye_lnf.jar**

## 拓展
只需要加入响应驱动并在**jdbcDrivers.xml**中配置即可。
